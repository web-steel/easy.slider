# Простой и красивый слайдер

| Событие клавиатуры | Действие                    |
|--------------------|-----------------------------|
| Стрелка вверх      | Переключение слайдера вверх |
| Стрелка вниз       | Переключение слайдера вниз  |
| Пробел             | Вкл/Выкл проигрывание видео |
| Enter              | Вкл/Выкл звук видео         |
